# Jim Kring Pi Extensions

Pi extensions for Jim Kring's coding workflow.

## Extensions

- `pr-status`: shows the current GitHub PR number and title in the Pi footer.
- `session-name`: exposes session naming as an LLM-callable tool.

## Install

From GitHub:

```bash
pi install git:github.com/jimkring/pi-extensions
```

For local development:

```bash
pi -e ~/Projects/pi/jimkring-pi-extensions
```

Or install the local checkout into user settings:

```bash
pi install ~/Projects/pi/jimkring-pi-extensions
```

## Development

```bash
npm install
npm run check
```
